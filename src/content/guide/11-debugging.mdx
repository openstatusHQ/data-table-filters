## Debugging

We use `react-scan` for performance debugging:

```
# .env.local
NEXT_PUBLIC_REACT_SCAN=true
```

TanStack Query devtools appear automatically in development. For table debug logs:

```
NEXT_PUBLIC_TABLE_DEBUG=true
```
