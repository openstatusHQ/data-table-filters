## Keyboard Shortcuts

The data table ships with built-in keyboard shortcuts to speed up common interactions like toggling filters, opening the command palette, or resetting state.

- `Cmd+K` — Toggle command palette
- `Cmd+B` — Toggle filter sidebar
- `Cmd+U` — Reset column state (order, visibility)
- `Cmd+J` — Toggle live mode
- `Esc` — Reset table filters
- `Cmd+.` — Reset element focus
